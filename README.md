<h3>
  主な機能
</h3>

<ul>
  <li>
    ログイン認証機能
  </li>
  <li>
    X APIを利用し自動投稿機能
  </li>
  
  <li>
    文字＋画像のポストの投稿（画像は4枚まで可能）
  </li>
  <li>
    文字＋画像の自動投稿（画像は4枚まで可能）
  </li>
  <li>
    自動投稿はスケジューリングされた時間で投稿される
  </li>
  <li>
    データベースの上から順に投稿していき、最終で1から投稿開始される
  </li>
  <li>
    保存されている投稿は削除アクションでGUS、DBから削除される
  </li>
</ul>

<h3>UI・UX</h3>

<ul>
  <li>
    保存された投稿の文字＋画像を一覧表示される
  </li>
  <li>
   一覧の画像はXのようにサイズを均等に配置しタップするとモーダルが出る
  </li>

</ul>
